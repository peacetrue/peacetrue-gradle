= 常见问题

== 在 Gradle 插件中执行其他插件

.内置插件直接执行：
`project.getPluginManager().apply(JavaLibraryPlugin.class)`

外部插件，需要配置依赖。如何配置依赖？

* 在 `BuildConventionPlugin` 通过 `buildscript` 无效
* 在 build.gradle 通过 implementation 'org.springframework.boot:spring-boot-gradle-plugin:2.7.0' 有效
